Windows OOBE 时开启管理员账户并跳过用户名设置

发表于 2023-12-23 分类于 随便写写

Win7&Win10 在 OOBE 设置用户名时开启 Adminnistrator 管理员账户并跳过 OOBE 设置。

准备

当 OOBE 进入 “键入用户名” 部分时，使用键盘 Shift + F10 快捷键打开 CMD 窗口。

步骤一 启用管理员账户

在 CMD 窗口中输入 lusrmgr.msc 启动 “本地用户和组”
选择 用户 双击选择 Adminnistrator 将 账户已禁用 (B) 取消勾选并确定。

步骤二 跳过 OOBE

返回 CMD 窗口输入 taskmgr.exe 启动 “任务管理器”
任务管理器中找到 msoobe 进程并将其结束。
