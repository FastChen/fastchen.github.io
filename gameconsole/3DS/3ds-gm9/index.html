3DS GodMode9 使用技巧记录

发表于 2023-03-17 更新于 2023-03-25 分类于 游戏机 ， 3DS

前言

买了一台 3DS 折腾，然后 WiiU 和 3DS 商店服务马上就关闭了，我下了很多官方的软件资源，为了防止丢失和后续重新安装，研究了一下 GodMode9 顺便吧折腾用到过的功能记录在这里。

GodMode9使用记录

本文GodMode9基于v2.1.1版本制作，如果你想用的功能没有，或有变化请确定你的版本是否等于或高于我的版本。

Home Menu 翻译对照

  • Poweroff system 关闭机器
  • Reboot system 重启机器
  • Title manager 游戏产品管理
  • Scripts… 脚本…
  • Payloads… gm9的插件?用于装载 .firm 文件的 (具体翻译待定…)
  • More… 更多…
    • SD format menu SD卡格式菜单
    • Bonus drive setup (未知)
    • Build support files 生成帮助文件
    • Set RTC date&time 设置 RTC [^1] 日期
    • Configure brightness 设置亮度
    • Calibrate touchscreen 校准触摸屏
    • System info 系统信息
    • Show ReadMe 查看”读我”文档

[^1]: 一般代指 实时时钟 (Real_Time Clock)

无卡开机 (将 Luma3DS 复制到 CTRNAND)

  1. 进入GM9 (按住 Start键 开机)
  2. 按 Home键 进入 Home menu
  3. 选择 Scripts… 按 A 确定
  4. 选择 GM9Megascript 按 A 确定
  5. 选择 Scripts from Plailect’s Guide 按 A 确定
  6. 选择 Setup Luma3DS to CTRNAND 按 A 确定
  7. 出现提示时，按 A 键继续
  8. 按 A 键 解锁 SysNAND (lvl1) 写入权限，并根据屏幕上的提示依次按下按键
  9. 按 A 键 完成

做这个可以让机器在tf卡移除或损坏的情况下继续开机进入系统，而不需要一张新的tf卡。

生成并导出 CIA 文件(游戏)

  1. 进入GM9 (按住 Start键 开机)
  2. 按 Home键 进入 Home menu
  3. 选择 Title manager 按 A 进入
  4. 选择 [A:] SD CARD 按 A 进入 (读取需要一些时间)
  5. 看上屏部分，通过十字键高亮到要生成 CIA 的游戏(软件)上，然后按 A 确定
  6. 下屏选择 Manage Title... 按 A 进入
  7. 选择 Build CIA (standard) 按 A 确定

然后等待进度条跑完，CIA 文件就会生成在 SD卡/gm9/out/ 文件夹里了

导出自制 GBA VC (Virtual Console) 游戏存档

导出部分 教程

这个是用于使用 New Super Ultimate Injector (NSUI) 这款软件制作的 GBA ROMs 安装版(.cia)存档导出，仅适用于GBA版，同时需要游戏已生成正确的存档。

  1. 进入GM9 (按住 Start键 开机)
  2. 看上屏并使用 ↓键(十字键) 选择 [S:] SYSNAND VIRTURAL 按 A 进入
  3. 上屏选中 agbsave.bin 按 A 下一步
  4. 看下屏菜单选择 AGBASVE options.... 按 A 下一步
  5. 选择 Dump GBA VC save 导出存档 按 A 下一步
  6. 提示 Savegame dumped to 0:/gm9/out. 按 A 下一步
  7. 导出完成，存档就存放到 SD 卡的 gm9 目录里 out 目录下。

导入部分 教程

导入前，先做一次导出，获取对应的存档名称。

  1. 将存档修改正确的名称 通常以 TitleID(一串数字).gbavc.sav 格式命名
  2. 把存档放到 SD 卡里，地方随意，能找到就行
  3. 进入GM9 (按住 Start键 开机)
  4. 上屏选择 [0:] SDCARD 进入
  5. 找到存档文件并选中，下屏提示 Y - COPY files 按 Y 复制
  6. 看上屏并使用 ↓键(十字键) 选择 [S:] SYSNAND VIRTURAL 按 A 进入
  7. 上屏选中 agbsave.bin 按 A 下一步
  8. 看下屏菜单选择 AGBASVE options.... 按 A 下一步
  9. 选择 Inject GBA VC save 按 A 下一步
  10. 显示 Savegame inject success. 就代表成功了，按 A 结束

如果没有导入成功

  1. 存档名字没有改对，推荐先生成一个存档，然后记录正确的名字。
  2. 复制文件的时候没有成功，成功复制文件的时候下屏会显示 [CLIPBOARD] 和已拷贝列表
  3. 检查游戏是否能正常存档/读档 (部分制作的 GBA.cia 不能正常使用存档功能)
